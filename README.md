# Сайт для заказа маршрутки

## Запуск и установка:

- Установка пакетов
    ````
    pip install -r requirements.txt
    ````

- Проведение миграций:
    ````
    python manage.py migrate
    ````

- Запуск приложения:
  ````
  python manage.py runserver
  ````

## Технологии, которые использовались в данном проекте:

- Django
- PostgreSQL
- HTML
